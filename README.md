# deepseek-期末大作业工作流
一．	项目简述：
1.	云游向导工作流：输入想要去的中国城市名和文风（输入示例：杭州 兴奋；上海 奇幻），就能生成一篇未来两三天的，结合实时新闻天气的虚拟图文游记。操作简单，响应速度快，结果稳定。
2.	运行条件：由于我是在coze上开发的，所以将文件导入coze里就能成功运行
3.	各节点的功能介绍
开始：用户输入city和writing_style
get_current_datetime插件：自动获取实时时间，以年-月-日 时-分-秒的字符串形式输出。
提取日期信息（代码节点）：从get_current_datetime插件中去除年月日信息分别放在year、month、day中
天气新闻获取（大模型节点）：大模型接收城市和日期信息，调用天气获取插件获取今后三天的天气信息，调用新闻获取插件，以城市名和日期为关键词进行检索
作家（大模型节点）：接收文风和天气新闻获取节点的信息，生成特定文风的虚拟游记。
画家的助手（大模型节点）：接收作家节点的游记，选择其中最生动的场景，用英语描述作为画家节点的提示词
画家（大模型节点）：用助手的提示词，调用生图插件，生成高质量的图片，输出该图片的url链接
Markdown形式输出（代码节点）：将游记和图片放在一起以markdown形式输出
结束：输出markdown文件
